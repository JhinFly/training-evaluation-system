# 培训满意度综合评价表

## 项目说明

这是一个培训满意度综合评价表的H5页面，用户可以通过手机扫码访问并填写评价表单，管理员可以查看和导出数据。

## 功能特点

1. **手机扫码访问**：生成二维码后，用户可以通过手机扫码直接打开评价表单
2. **响应式设计**：适配各种手机屏幕尺寸
3. **一键提交**：用户填写完成后可以一键提交评价
4. **实时数据查看**：管理员可以登录后台查看提交的评价数据
5. **自动统计**：系统会自动统计评价数据
6. **导出Excel**：管理员可以导出评价数据为Excel格式

## 部署方法

### 1. 前端部署

将`index.html`文件部署到任何静态网站托管服务，例如：

- **GitHub Pages**：免费托管，适合小流量应用
- **Vercel**：免费托管，支持自动部署
- **Netlify**：免费托管，支持自定义域名

### 2. 后端实现

由于需要实现数据存储和管理功能，我们可以使用以下免费服务：

#### 方案一：使用Google Forms
1. 创建一个Google Form，添加与评价表对应的字段
2. 将Google Form的响应链接嵌入到`index.html`中
3. 管理员可以通过Google Forms后台查看和导出数据

#### 方案二：使用Firebase
1. 创建一个Firebase项目
2. 启用Firebase Realtime Database或Firestore
3. 添加Firebase认证功能，用于管理员登录
4. 修改`index.html`中的提交逻辑，将数据发送到Firebase
5. 创建一个简单的管理员页面，用于查看和导出数据

#### 方案三：使用SheetDB
1. 创建一个Google Sheet
2. 使用SheetDB将Google Sheet转换为API
3. 修改`index.html`中的提交逻辑，将数据发送到SheetDB
4. 管理员可以直接在Google Sheet中查看和导出数据

## 二维码生成

1. 部署前端页面后，获取页面URL
2. 使用任何二维码生成工具（例如：https://www.qr-code-generator.com/）生成二维码
3. 将生成的二维码保存并分发给用户

## 自定义修改

1. 可以根据实际需求修改`index.html`中的表单字段
2. 可以调整CSS样式以适应品牌风格
3. 可以根据选择的后端方案修改提交逻辑

## 注意事项

- 确保部署的网站可以被外部访问
- 选择合适的后端方案，根据预期的访问量和数据量
- 定期备份评价数据，以防数据丢失
- 遵守相关数据隐私法规，保护用户信息
